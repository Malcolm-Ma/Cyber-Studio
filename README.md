## Project Structure

```
.
|-- solution
    |-- service-app

```

### web-app

Front end of the project.

### service-app

Back end of the project. 

Connect MongoDB.

## Get Started

We need to start the project via service-app and web-app separately. 

```shell
cd solution
```

### Start service-app

```shell
cd service-app
npm install
npm run initdb
npm run start
```

### Start web-app

```shell
cd web-app
npm install
npm run start
```
